Simple Facebook Feeds Display through shortcode.

== Description ==

A simple and easy to use plugin to display your facebook feeds in your wordpress blog and website.

= Key Features =

* Easy to use.
* Supports all browsers.
* Display feed through widget.
* Mobile Friendly.
* Can display Facebook feeds with shortcode.
* User can limit the feeds to display.
* Show feeds "only from page owner" or "Owner + Other People" or "Other People".
* Set width & height of feeds container.
* Hide / Show following features.
    .  Author name and avatar.
    .  Post Date.
    .  Links, Videos, Photos.
    .  Post Message.
    .  View on Facebook.
    .  Page Cover Photo.
    .  Like us Button.

= How to use =

* Simple Usage: You can display feeds on any post or page you like with a simple short code `[simple-facebook-feed]`
* Limit Feeds: You can use `[simple-facebook-feed limit="10"]` to limitise the facebook feeds.

== Installation ==

= Manual Install =
1. Download the package.
2. Upload plugin folder `/simple-facebook-feeds` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
= Install through WordPress =
1. Go to Plugins > Add New.
2. Search for `simple facebook feeds`
3. Install it and enjoy :)
